# AdBlockList

This repo consists of the one (**large!**) list of domains for use in any DNS/ad blocker applications.

## :question: How to use this ad blocklist?

 1. Copy RAW link to _hosts.txt_ file.
 2. Import it to your ad blocker app.
 3. Enjoy!

## :blue_book: References
Based on [official Brave list](https://github.com/brave/adblock-resources), but includes also lists from the various authors. All original links are present in _sources.json_ file.
